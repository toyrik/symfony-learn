# Описание

Контейниризированное приложение на symfony 

Для запуска контейнеров

```bash
# С пересборкой контейнеров
make init

# Без пересборки контейнеров
make up
```

Для останова контейнеров

```bash
make down
```

Для выполнения команд в контейнере

```bash
docker-compose run --rm php-cli <команда>
```
